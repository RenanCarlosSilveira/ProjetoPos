Estruturação

Projeto feito com as seguintes ferramentas:

  • Java - Versão 17
  • Tomcat - Versao 9
  • SpringBoot - Versão 3.2 + Spring Security Default
  • MySQL

Repositorio:

URL: https://github.com/RenanCarlosSilveira/ProjetoPos


Banco de dados:

URL: jdbc:mysql://localhost:3306/ProjetoPos?createDatabaseIfNotExist=true

Schema: ProjetoPos

OBS: Talvez precise criar o schema no servidor do banco, tabela e outras coisas não precisa

Porta: 3306


Servidor:

Tomcat

Porta: 8181

Path da aplicação: http://localhost:8181/projetopos/


Subir a aplicacao:

Java 17 instalado

Variavel de ambiente Path com caminho do Java Correspondente

Executar terminal na pasta e arquivo com "java -jar ProjetoPos.jar"

Usuario padrão de login do security é user

A senha será gerada no console em Using generated security password: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
